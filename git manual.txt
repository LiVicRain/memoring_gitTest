git init  : 컴퓨터 폴더에 깃 생성

git remote add origin https://github.com/LiVicRain/Testing_msi.git : https://github.com/LiVicRain/Testing_msi.git이란 넷상의 주소와 깃을 연결

push : 는 깃 스토리지로 자료를 올리는것

pull : 자료를 가져오는것

push 전에 pull이 있다.


git pull origin master : 지정해 놓은 Repository에서 자료를 끌어온다. (모든 자료를 가져온다.)

git status : 현재 Git과의 상태를 Check해준다.

git add . : 변경된 파일들을 모두 Tracking 해준다.

git commit -m "message" : 인식 할 수 있는 commit을 달아준다.

git push origin +master : 실제로 Git에 push 해준다.
